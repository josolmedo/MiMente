Make Directory): Se utiliza para crear nuevos directorios o carpetas desde la terminal.

**Uso Principal del comando mkdir:**

- **Crear un directorio:** `mkdir nueva_carpeta` crea una carpeta en la ubicación actual.

- **Crear múltiples directorios:** `mkdir carpeta1 carpeta2 carpeta3` genera varios directorios simultáneamente.

- **Crear subdirectorios (p):** `mkdir -p /ruta/a/carpeta/anidada` permite crear directorios padre si no existen, evitando errores.

- **Establecer permisos (m):** `mkdir -m 755 carpeta` crea el directorio con permisos específicos (en este ejemplo, 755).    

- **Verificar creación (v):** `mkdir -v carpeta` muestra un mensaje confirmando la creación del directorio.
